This project (EvoForm or function-graph-evolution) is authored by fwjmath under GPLv3.

It needs SDL to compile.

To run, make sure you have a gallery/ directory with a cnt.txt file containing 0 in the same directory of the application.

Space to select/unselect. Press e to start evolution when 6 pictures are selected. Press Enter to enter/exit zoomed mode. In the zoomed mode, press s once to save the current graph as bmp under gallery/.
